# alexmcc.dev

My personal site. It's a static HTML page.

I build distributed systems and ML-powered credit decisioning experiences for a living. This site displays my name and some links. The complexity budget was spent accordingly.

## Stack

- `index.html` — the website
- `style.css` — makes it look like I tried
- `script.js` — minimal behavior, no opinions

No build step. No dependencies. No configuration files for your configuration files.

## Running locally

```sh
open index.html
```

Or if you need a server:

```sh
python3 -m http.server
```

## Deploying

Four-line GitHub Actions workflow. Pushes to GitHub Pages on merge to main. You can read the whole thing in under a minute.
